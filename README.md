# MD: Collapsed Russia

A modification for **Millennium Dawn: A Modern Day Mod** (Hearts of Iron 4) that adds a scenario where Russia collapses into multiple independent states.

## 📋 Description

This mod presents an alternative history in which the Russian Federation disintegrates into separate regional states. The mod adds new countries, focus trees, decisions, and Russian language localization.

## ✨ Features

- **New Countries**: Various regional states added across the territory of the former Russia (Amur, Arkhangelsk, Chelyabinsk, Jewish AO, Irkutsk, Ivanovo, Kostroma, Kurgan, Kemerovo, Khabarovsk, Zabaykalye, Primorye, Kamchatka, Sakhalin, Omsk, Novosibirsk, Krasnoyarsk, Samara, Saratov, Orenburg, Nizhny Novgorod, Penza, Tomsk, and others)
- **Russian Localization**: Full Russian language support for new countries and content
- **National Spirits**: Unique spirits for certain countries (e.g., Vologda)
- **Decisions**: New decisions for Russia
- **Focus Trees**: National focus trees for individual countries

## 🔧 Requirements

- **Hearts of Iron 4** (version 1.17.x)
- **Millennium Dawn: A Modern Day Mod** (required dependency)
